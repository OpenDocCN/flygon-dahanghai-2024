#### 4.1.6 关键帧

视频剪辑中，关键帧特指在时间轴上选择的特定帧，它们代表了视频中的重要瞬间或转折点，一般用来标记视频中重要的变化或状态转换。

依靠关键帧，我们能实现画面从小到大、声音由弱变强、移动水印等操作。

### 5.1 什么是混剪和批量混剪
